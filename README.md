# Drum Practice App

A mobile-friendly web app for practicing drum patterns with multiple instruments.

## Features

- Create and manage songs
- Record beat patterns for multiple instruments (Surdo 1, Surdo 2, Dobra, Repi, Snare)
- Grid-based beat editor with 16th note subdivisions
- Tap recording along with click track
- Solo/mute individual instruments
- Adjustable BPM per song
- Organize patterns into named subsections (Intro, Verse, Chorus, etc.)
- View-only mode for playback
- Edit mode protected by password

## Usage

Visit the hosted page and browse songs. Click on any song to play its subsections.

To edit patterns, click "Login" and enter the password.
